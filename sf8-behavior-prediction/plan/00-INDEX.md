# SF8 Planning Index

This folder stores the approved planning artifacts for `SF8 - AI-based Customer Behavior Prediction (New Customers)`.

## Files

- `01-DECISIONS-SUMMARY.md`
  Approved decisions summary for scope, governance, scoring, submission, and execution.
- `02-CONTROL-FILES-SPEC.md`
  Detailed spec for the 6 control files.
- `03-BATCH1-CANDIDATES-SPEC.md`
  Batch 1 candidate rules and assembly constraints.
- `04-MASTER-PLAN.md`
  Full implementation master plan by session.
- `05-EXECUTION-BOARD-TEMPLATE.md`
  Standard execution board template.
- `06-EXECUTION-BOARDS-01-06.md`
  Execution boards for Sessions 01-06.
- `07-EXECUTION-BOARDS-07-12.md`
  Execution boards for Sessions 07-12.
- `08-EXECUTION-COMPLETION-RULES.md`
  Concrete completion rules so sessions are not marked done at spec-only state.

## Usage order

1. Read `01-DECISIONS-SUMMARY.md`
2. Read `02-CONTROL-FILES-SPEC.md`
3. Read `03-BATCH1-CANDIDATES-SPEC.md`
4. Read `04-MASTER-PLAN.md`
5. Read `08-EXECUTION-COMPLETION-RULES.md`
6. Use `05-EXECUTION-BOARD-TEMPLATE.md`, `06-EXECUTION-BOARDS-01-06.md`, and `07-EXECUTION-BOARDS-07-12.md` during execution

## Important constraints

- These files contain only approved planning content.
- They do not approve any unresolved business content such as final partner names, final product package names, or final candidate datasets.
- Any future candidate content must still pass the approval gates captured in these files.
