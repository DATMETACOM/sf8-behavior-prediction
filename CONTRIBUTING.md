# Contributing to Shinhan PoC

This is a private repository for Qwen AI Build Day 2026 hackathon.

## Setup

1. Clone the repository
2. Copy `.env.example` to `.env`
3. Add your Qwen API key
4. Run `npm install` in project directories

## Projects

- `sb10-branch-prediction/` - Branch Traffic Prediction
- `sf8-behavior-prediction/` - Customer Behavior Prediction

## Questions?

Contact the team via GitLab issues.
